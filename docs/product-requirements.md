# VINYL FIND 제품 요구사항

이 문서는 데모 앱 구현 시 기준으로 사용하는 제품 요구사항이다. 데모는 실제 서비스 흐름을 보여주되, 거래와 사용자 데이터는 모두 목업 데이터로 처리한다.

## 1. LP 검색

앨범, 곡, 아티스트 이름으로 LP 데이터베이스를 검색한다.

- 앨범에 LP 판본이 있으면 발매 정보, 판매처, 중고 거래 정보를 확인할 수 있다.
- 앨범이 바이닐로 발매되지 않았다면 해당 아티스트의 다른 LP 발매 앨범을 안내한다.
- 곡이 수록된 앨범 가운데 LP 판본이 있는 앨범을 모두 제공한다.
- 아티스트를 검색하면 해당 아티스트의 LP 발매 앨범을 모두 제공한다.

## 2. 발매 정보와 시세

같은 앨범에 여러 바이닐 판본이 있으면 앨범 상세 안에서 판본을 구분해 제공한다.

- 앨범 발매일, LP 판매 기간, 현재 판매 상태, 생산 수량을 제공한다.
- 상시 판매 중인 상품은 구매 가능한 국내외 판매처와 가격을 비교한다.
- 단종 및 한정반은 VINYL FIND 검수 재고와 최근 체결 거래를 중심으로 제공한다.
- 해외 판매처 상품은 여러 구매자의 주문을 모아 배송 단가를 낮추는 공동구매로 연결할 수 있다.

## 3. 앨범 정보와 리뷰

수록곡과 앨범 배경 정보를 위키 형태로 제공하고 사용자가 수정과 추가에 기여할 수 있게 한다. 인기 앨범은 서비스가 기본 정보를 먼저 구성한다. 사용자는 별도의 평가와 평론을 작성할 수 있다.

## 4. 중고 거래

### 정보 제공

- 다른 사용자의 구매 주문 조건과 희망 가격
- 판매 중인 사용자 매물과 VINYL FIND 검수 재고
- 서비스 내 최근 체결 거래
- 외부 중고 거래 데이터를 참고한 기준 가격과 예상 가격

### 즉시 구매

현재 판매 중인 제품과 VINYL FIND 검수 재고를 목록으로 보여준다. 구매자는 상태와 가격을 비교해 하나를 선택한 뒤 바로 구매한다.

데모 동작:

1. 상세 화면에서 `즉시 구매`를 누른다.
2. 현재 구매 가능한 재고를 가격순으로 확인한다.
3. 하나를 선택하고 구매를 확정한다.
4. `즉시 구매가 완료되었습니다.` 상태를 보여준다.

### 구매 조건 설정

구매자는 다음 조건을 지정해 구매 예수 주문을 등록한다.

- 개봉 또는 미개봉
- 바이닐 상태 최소 등급
- 자켓 상태 최소 등급
- 청음 상태 최소 등급
- 최대 구매 가격

현재 재고 중 조건에 맞는 상품이 있으면 가장 저렴한 상품을 보여주고 구매 주문을 체결한다. 일치하는 재고가 없으면 대기 주문을 등록하고 이후 조건에 맞는 상품이 입고될 때 연결한다.

데모 동작:

1. 상세 화면에서 `구매 조건 설정`을 누른다.
2. 개봉 여부, Goldmine 등급, 최대 가격을 지정한다.
3. 조건에 맞는 재고가 있으면 가장 저렴한 상품과 함께 `구매 주문이 체결되었습니다.` 상태를 보여준다.
4. 조건에 맞는 재고가 없으면 `구매 대기 주문이 등록되었습니다.` 상태를 보여준다.

### 판매

판매자는 VINYL FIND에만 제품을 판매한다. 사진 업로드를 통해 예상 등급과 견적을 먼저 받고, 제품 전달 후 실제 검수 결과와 최근 구매 수요를 바탕으로 최종 금액을 지급받는다.

### 상태 평가와 가격 책정

Goldmine Grading System을 사용해 매니아층도 신뢰할 수 있는 일관된 상태 기준을 제공한다.

- 미개봉: 포장 상태와 자켓 손상을 평가한다.
- 개봉: 자켓 상태, 바이닐 상태, 청음 상태를 각각 평가한다.
- 가격은 발행 정보, 한정판 여부, 인기도, 외부 중고 거래 데이터, 서비스 내 수요를 종합해 책정한다.
- 기준 참고: [Discogs - How To Grade Items](https://support.discogs.com/hc/en-us/articles/360001566193-How-To-Grade-Items)

### 검수 시스템

사람이 확인할 항목:

- 프레싱 정보와 판본 식별
- 개봉 여부
- 구성품 유무

AI와 일관된 촬영·녹음 환경으로 확인할 항목:

- 자켓 변색, 구겨짐, 찢김
- 바이닐 휨, 먼지, 곰팡이, 얼룩, 긁힘
- 전체 재생 녹음 기반 청음 문제

## 5. 커뮤니티

LP 발매 정보, 리뷰, 추천, 기기 추천과 자유 주제를 다루는 커뮤니티를 제공한다. 익명 글쓰기, LP 자랑, 추천, 평가, 정보 가이드, 교환, 경매를 지원한다.

## 6. 바이닐 디깅

오프라인 중고 매장의 진열 방식을 온라인에서 탐색할 수 있게 한다. 재고를 한 장씩 꺼내 보며 앨범 배경 정보와 청음 링크를 바로 확인할 수 있다.

## 7. 프로필과 소셜

사용자는 소유한 LP와 청음한 LP를 프로필에 등록하고 평가, 평론, 컬렉션을 공유할 수 있다.
